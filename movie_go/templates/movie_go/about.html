This website has been built as a part of a student assignment. The dataset for the movies has been taken from Kaggle The images for all the movies have been taken from the TMDB API

This website was built using Django, which is an open-source framework for building web based applications in Python.
Here are some of the features of this application.

  • 1. A movies page that lists out the top-rated movies and allows users to search for a movie based on name and genre.
  • 2. A zone details page that lists the zones available to watch the selected movie.
  • 3. An extensive authentication system that allows the users to signup, login and logout securely.
  • 4. A dashboard for staff users to monitor the orders and customers count everyday.
  • 5. An orders page for the customers to view the orders they have made.
